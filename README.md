## Getting Started
### Prerequisites
- [Install NodeJS ](https://nodejs.org/en/download)
- Install NPM
```
npm install npm@latest -g
```
### Installation
1. Clone the repository
2. Install NPM packages
```
$ cd client
$ npm i --force
```
```
$ cd server
$ npm i
```

## Usage
1. Run the React scripts
```
$ cd server
$ node index.js
```
```
$ cd client
$ npm start
```
